# Semana 3 — Quiz Auditoría y Refactorización

##  Descripción del Proyecto

Este proyecto corresponde al Quiz de la Semana 3, enfocado en la auditoría de código, identificación de vulnerabilidades y construcción de un ADR (Architecture Decision Record).

El sistema implementa un módulo básico de autenticación con funcionalidades de:

- Registro de usuarios
- Login
- Endpoint de health check

---

##  Tecnologías utilizadas

- Java 17+
- Spring Boot
- Maven
- Base de datos relacional (H2 / MySQL)
- REST API
